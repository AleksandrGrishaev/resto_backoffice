# Solar Kitchen — Loyalty Program

_Финальная спецификация, март 2026_

---

## Два блока программы

```
БЛОК 1: ФИЗИЧЕСКАЯ КАРТОЧКА        БЛОК 2: ЦИФРОВОЙ ПРОФИЛЬ
--------------------------------------------------------------
Механика:  Штампы -> бесплатно      Баллы = деньги (1 балл = 1 IDR)
Кто:       Новый клиент / турист   Зарегистрированный через Telegram
Статус:    Номер карточки           Уровень с cashback %
Цель:      Точка входа             Удержание, частота, LTV

Переход Блок 1 -> Блок 2:
  Клиент называет номер карты ->
  кассир вводит номер в POS ->
  штампы конвертируются в баллы (+10% бонус) ->
  клиент регистрируется в Telegram-боте ->
  присваивается уровень Member ->
  физическая карточка закрывается
```

Оба блока полностью настраиваются через Admin-интерфейс без разработчиков.

---

## Кто может участвовать

### Блок 1 — Физическая карточка

**Кто получает:**

- Любой новый гость при первом визите — автоматически
- Турист, который не хочет регистрироваться
- Гость без Telegram или нежелающий регистрироваться

**Условия:**

- Выдается кассиром при любом заказе
- Не требует регистрации, имени, телефона
- Каждой карточке присваивается уникальный порядковый номер: 001, 002...

**Кто НЕ получает новую карточку:**

- Клиент с действующим цифровым профилем
- Клиент у которого уже есть активная карточка

**Срок действия:**

- Каждый штамп живет 90 дней с момента начисления
- Штампы старше 90 дней сгорают автоматически

---

### Блок 2 — Цифровой профиль

**Кто получает:**

- Клиент который зарегистрировался через Telegram-бот
- Клиент конвертировавший физическую карточку
- Текущие 30-40 экспатов — вручную при запуске, уровень VIP

**Условия:**

- Регистрация через Telegram-бот
- Один аккаунт Telegram = один профиль, дубли не создаются

**Уровни и cashback:**

| Уровень | Порог трат за 90 дней | Cashback | Кто стартует                        |
| ------- | --------------------- | -------- | ----------------------------------- |
| Member  | 0 — при регистрации   | 5%       | Любой новый                         |
| Regular | TBD                   | 7%       | Автоматически                       |
| VIP     | TBD                   | 10%      | Экспаты при запуске + автоматически |

Пороги для Regular и VIP настраиваются в Admin. Пересчет автоматический ежедневно по скользящему окну 90 дней.

**Правила уровней:**

- Деградация максимум на 1 уровень за период
- Member — нижний уровень, не падает
- При повышении и понижении — уведомление в Telegram

**Кто НЕ участвует в Блоке 2:**

- Анонимный гость — остается на карточке
- Гость который отказался — предлагаем карточку

## Блок 1 — Физическая карточка

### Учет карточек

```
Выдача:
  Кассир -> "Новая карточка" в POS
  Система присваивает следующий номер (001, 002...)
  Кассир вписывает номер на карточку маркером
  (или карточки заранее напечатаны с номерами)

Каждый визит:
  Клиент называет номер карточки
  Кассир вводит номер -> видит: штампов сейчас, дата последнего визита
  Система рассчитывает штампы за текущий чек
  Фиксируется: дата, сумма чека, штампов начислено

Конвертация:
  Клиент называет номер карточки + регистрируется в боте
  Кассир вводит номер -> система показывает штампы
  Кассир привязывает Telegram аккаунт к карточке
  Баллы начисляются автоматически, карточка закрывается
```

### Параметры (настраиваются в Admin)

| Параметр            | Значение по умолчанию                  |
| ------------------- | -------------------------------------- |
| Штампов на карточке | 15                                     |
| Порог одного штампа | 80 000 IDR (с налогами)                |
| Формула             | `floor(сумма_чека / 80 000)`           |
| Срок жизни штампа   | 90 дней с момента начисления           |
| Цикл                | После 15-го штампа — сброс, новый цикл |

### Награды (настраиваются в Admin)

| Штампов | Категория блюд          | Потолок скидки |
| ------- | ----------------------- | -------------- |
| 5       | Напитки                 | 40 000 IDR     |
| 10      | Завтраки и легкие блюда | 75 000 IDR     |
| 15      | Любое блюдо из меню     | 100 000 IDR    |

### Правила начисления

```
Чек 70 000 IDR  -> 0 штампов
Чек 80 000 IDR  -> 1 штамп
Чек 140 000 IDR -> 1 штамп
Чек 160 000 IDR -> 2 штампа
Чек 280 000 IDR -> 3 штампа

База: сумма чека с налогами 15%
Несколько визитов в день: суммируются без ограничений
```

### Применение награды

- Кассир вводит номер карточки -> POS показывает активную награду
- Система показывает позиции из чека подходящие под категорию
- Скидка = фактическая стоимость позиции, не выше потолка
- Пример: награда "завтрак до 75k", клиент взял боул за 65k -> скидка 65 000 IDR

### Экономика

```
Типичный заказ кофе + блюдо с налогами: ~138 000 IDR -> 1 штамп

5 штампов:  ~550 000 IDR  -> free напиток 40 000 = 7.3%
10 штампов: ~1 100 000    -> free завтрак 75 000 = 6.8%
15 штампов: ~1 650 000    -> free блюдо 100 000  = 6.1%

Суммарно за цикл: 215 000 / 1 650 000 = 13% cost of loyalty
```

## Блок 2 — Цифровой профиль (баллы)

### Механика баллов

```
1 балл = 1 IDR. Всегда.

Начисление:
  cashback % от суммы чека с налогами, после списания
  Пример: чек 138 000 IDR, Member (5%) -> +6 900 баллов

Списание:
  Как деньги — клиент решает когда и сколько
  До 100% суммы чека
  Кассир предлагает, клиент подтверждает

Срок жизни: 90 дней с момента начисления каждого балла
Уведомление о сгорании: за 7 дней в Telegram
```

### Автоматический пересчет уровней

```
Окно: последние 90 дней (скользящее, обновляется ежедневно)

Логика:
  Сумма чеков клиента за 90 дней >= порог VIP   -> уровень VIP
  Сумма чеков клиента за 90 дней >= порог Regular -> уровень Regular
  Сумма чеков ниже порога Regular               -> уровень Member

Деградация: максимум -1 уровень за пересчет
  Пример: был VIP, не дотянул до VIP -> Regular (не Member)

Уведомления:
  Повышение -> поздравление в Telegram
  Понижение -> уведомление с мотивацией
```

### Конвертация физической карточки

```
Баллы = (штампы * 80 000) * cashback_уровня + 10% бонус

Пример: 7 штампов, Member (5%)
  560 000 IDR * 5% = 28 000 + 2 800 бонус = 30 800 баллов

Пример: 7 штампов, VIP (10%)
  560 000 IDR * 10% = 56 000 + 5 600 бонус = 61 600 баллов
```

## Личный кабинет клиента

Клиент взаимодействует через **Telegram-бот**. Бот может открывать **веб-кабинет** (Mini App или внешняя ссылка).

### Что доступно в боте

- Текущий уровень и cashback %
- Баланс баллов и дата ближайшего сгорания
- Прогресс до следующего уровня (сколько потратить за 90 дней)
- История последних визитов: дата, сумма, начислено, списано
- Уведомления: начисление после каждого визита, сгорание за 7 дней, смена уровня

### Что доступно в веб-кабинете

Идентично боту. Открывается по ссылке из бота или от кассира — без логина, по токену в URL.

## Admin-интерфейс — что настраивается

### Программа 1: Карточки

- Количество штампов в цикле
- Порог одного штампа (IDR)
- Список наград: штамп -> категория блюд -> потолок скидки
- Срок жизни штампов (дней)
- Управление карточками: список по номерам, статусы, история визитов

### Программа 2: Cashback

- Список уровней: название, cashback %, порог трат за 90 дней
- Правило деградации (максимум -N уровней за период)
- Срок жизни баллов (дней)
- Бонус за конвертацию карточки (%)

## POS — сценарии

### Клиент с физической карточкой

1. Называет номер карточки -> кассир вводит
2. POS показывает: штампов сейчас, дата последнего визита
3. Кассир вводит сумму -> система добавляет штампы
4. Если есть активная награда -> список подходящих позиций из чека -> скидка

### Клиент с цифровым профилем

1. Называет имя или показывает бот -> кассир находит профиль
2. POS показывает: уровень, баланс, начислится сейчас
3. Кассир предлагает списать баллы -> клиент решает
4. Оплата -> cashback начисляется, уведомление в Telegram

### Регистрация нового клиента

1. "Новый клиент" в POS -> выбор: Telegram
2. Клиент открывает бот -> вводит имя -> профиль создан
3. Кассир привязывает профиль в POS -> уровень Member
4. Если есть карточка -> вводим номер -> конвертация баллов

### Выдача новой карточки

1. "Новая карточка" в POS
2. Система присваивает следующий номер
3. Кассир вписывает номер на карточку и выдает гостю

## Перевод текущих 30-40 экспатов

**День запуска:**

1. Кассир открывает "Конвертация" в POS
2. Вводит номер физической карточки клиента
3. Клиент регистрируется в боте прямо у стойки
4. Кассир привязывает Telegram -> уровень VIP вручную
5. Баллы начисляются автоматически + 10% бонус
6. Клиент видит баланс в боте мгновенно

**Через 30 дней:** физические карточки убираются с бара.
